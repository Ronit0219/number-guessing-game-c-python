# Number Guessing Game (C & Python)

A beginner project created while learning programming fundamentals.

## Features

### Version 1
- Random number generation
- High/Low hints
- Input validation

### Version 2
- Difference hints
- Play again functionality

### Version 3
- Limited attempts

### Version 4
- Difficulty levels
  - Easy (1-50, 12 attempts)
  - Medium (1-100, 10 attempts)
  - Hard (1-200, 8 attempts)

## Languages
- C
- Python
